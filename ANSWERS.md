# What are PropTypes used for? Please describe why it's important to type check our data in JavaScript.
PropTypes are used to check the type of data that is being passed into components. This is important in JavaScript
because it is a loosely typed language and does not require specific data types for arguments. This can be an issue if the argument that is being padded in is being operated on in a type-specific way.
# Describe a life-cycle event in React?
A react life-cycle event is a method that is specific to the Component class in react. These methods used to keep track of or manipulate a component based on the life-cycle. 
# Explain the details of a Higher Order Component?
Higher order components are components that take components as an argument and can return components as well.
# What are three different ways to style components in React? Explain some of the benefits of each.
Three different ways to style components in react are with CSS, inline styling, and with ReactStrap or Styled components. 
Styling with CSS is beneficial for keeping your styles abstract and reusable in react as well as HTML. The advantage of inline styling a react component is testing. This is the most specific way to select and style a DOM element, making pinpointing a bug easy work. Styled components are extremely convenient because the can be written in the JS files.
